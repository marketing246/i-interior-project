# AI Дизайнер Интерьеров

Это приложение позволяет пользователям загружать фотографию своей комнаты, описывать желаемый стиль и получать четыре уникальных концепции дизайна интерьера, созданных с помощью Gemini 2.5 Flash Image.

## Финальная инструкция по развертыванию (Самый простой способ)

Этот метод использует встроенную функцию **"Save to Github"** и сервис **[Vercel](https://vercel.com)** для максимально быстрой и простой публикации вашего приложения.

### Шаг 1: Сохраните код на GitHub

1.  **Нажмите кнопку "Save to Github"** в интерфейсе среды разработки.

2.  **Подключите ваш аккаунт GitHub**.

3.  **Придумайте имя для репозитория**.
    *   **ВАЖНО**: GitHub не позволит вам использовать имя, которое уже занято. Если вы видите ошибку "repository already exists", просто **введите любое другое уникальное имя**.
    *   **Рекомендация**: Используйте имя `i-interior-project`, как мы и договорились.
    
4.  Нажмите **"Create"**. Через несколько секунд все ваши файлы будут сохранены в новом репозитории на GitHub.

### Шаг 2: Развертывание на Vercel

1.  **Зарегистрируйтесь на Vercel**: Перейдите на [Vercel.com](https://vercel.com) и зарегистрируйтесь, используя свой аккаунт GitHub.

2.  **Импортируйте проект**:
    *   В вашей панели управления Vercel нажмите "Add New..." -> "Project".
    *   Найдите и выберите ваш **новый** репозиторий (`i-interior-project`) и нажмите "Import".

3.  **Настройте проект**:
    *   Vercel автоматически определит, что это проект на Vite, и выставит все настройки правильно. **Вам ничего не нужно менять.**
    *   Раскройте секцию **"Environment Variables"** (Переменные окружения).
    *   Добавьте новую переменную:
        *   **Name**: `API_KEY`
        *   **Value**: Вставьте сюда ваш API-ключ от Google Gemini.
    *   Нажмите "Add".

4.  **Разверните приложение**:
    *   Нажмите кнопку **"Deploy"**.

### Шаг 3: Готово!

Подождите 1-2 минуты, пока Vercel соберет и развернет ваш проект. Вы получите рабочую ссылку на ваше приложение. **Теперь вы можете делиться этой ссылкой с кем угодно!**
